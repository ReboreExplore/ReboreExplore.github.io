  • Institut für Automatisierungstechnik und Softwaresysteme, Universität Stuttgart

  • Community Climate Center, Azim Premji University

    (May 2022 - Oct 2022) - Research Intern

    • Supervisor: Prof. Santonu Goswami
    • Project Title: Deep learning based flood mapping: A Case Study from a flood-vulnerable state in the Northeastern region of India
    • Project Overview :In this work we created a deep learning-based flood model that can predict the flood vulnerability of a region based on various spatial inputs, like slope, elevation, historical flood data, etc. These models make use of convolutional layers which are able to extract the spatial information based on the inputs we provide. Unlike the hydrodynamic models, our deep learning-based approach, which uses the ResNet architecture as the backbone, attempts to reduce the inference time of flood prediction for a given set of inputs.
    • Poster Presentation : This project was presented as a poster in the American Geophysical Union (AGU) Fall meeting 2022. Poster
  • Community Climate Lab (Feb 2022 - Oct 2022) - Research Intern
    • Supervisor: Prof. Santonu Goswami
    • Project Title: Mapping Biogeophysical Parameters for Indian Cities Using Long-Term Satellite Data and Geospatial Technologies

    • Project Overview : In this work, we have carried out an analysis to characterize the long-term changes in biophysical parameters such as vegetation, water bodies and built-up area for any specified target city or region, given the geographical boundaries. The dataset was obtained from the satellite data of Landsat 7. More about the project is available here.The final goal of the project aimed at building an automated pipeline which can now process long-term remotely sensed data and plot various biophysical indices eventually making the process of preliminary investigations of biophysical parameters fast by manifolds as well as efficient.

    • Poster Presentation : This project was presented as a poster in the American Geophysical Union (AGU) Fall meeting 2022. Link to the poster : Poster
