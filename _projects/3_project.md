This project is done as a part of my Research Assistantship at the Computational Cognitive Lab at the University of Stuttgart. The project aims to develop a pipeline to store, analyse and convert EEG data to BIDS format. We are developing a open source python package called LSLAutoBIDS.

You can find more about the package [here](https://github.com/s-ccs/LSLAutoBIDS)
