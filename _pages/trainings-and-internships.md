- **Community Climate Center, Azim Premji University** 

    (May 2022 - Oct 2022) - _Research Intern_
    - _Supervisor_: Prof. Santonu Goswami
    - _Project Title_: Deep learning based flood mapping: A Case Study from a flood-vulnerable state in the Northeastern region of India
    - _Project Overview_ :In this work we created a deep learning-based flood model that can predict the flood vulnerability of a region based on various spatial inputs, like slope, elevation, historical flood data, etc. These models make use of convolutional layers which are able to extract the spatial information based on the inputs we provide. Unlike the hydrodynamic models, our deep learning-based approach, which uses the ResNet architecture as the backbone, attempts to reduce the inference time of flood prediction for a given set of inputs.
    - _Poster Presentation_ : This project was presented as a poster in the American Geophysical Union (AGU) Fall meeting 2022. [Poster](https://agu2022fallmeeting-agu.ipostersessions.com/Default.aspx?s=CC-76-83-B4-7E-C7-A7-9F-86-ED-28-B4-75-4F-F9-56)


- **Community Climate Lab** (Feb 2022 - Oct 2022) - _Research Intern_
    - _Supervisor_: Prof. Santonu Goswami
    - _Project Title_: Mapping Biogeophysical Parameters for Indian Cities Using Long-Term Satellite Data and Geospatial Technologies
    - _Project Overview_ : In this work, we have carried out an analysis to characterize the long-term changes in biophysical parameters such as vegetation, water bodies and built-up area for any specified target city or region, given the geographical boundaries. The dataset was obtained from the satellite data of Landsat 7. More about the project is available [here](https://github.com/ReboreExplore/Spectral-Indices-Analysis-using-geospatial-frameworks).The final goal of the project aimed at building an automated pipeline which can now process long-term remotely sensed data and plot various biophysical indices eventually making the process of preliminary investigations of biophysical parameters fast by manifolds as well as efficient.

    - _Poster Presentation_ : This project was presented as a poster in the American Geophysical Union (AGU) Fall meeting 2022. Link to the poster : [Poster](https://agu2022fallmeeting-agu.ipostersessions.com/default.aspx?s=C5-13-53-03-34-BA-5F-20-3A-45-5A-9B-62-96-F3-2E&guestview=true)
